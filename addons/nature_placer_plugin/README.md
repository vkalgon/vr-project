# Nature Object Placer Plugin

Плагин для размещения объектов природы на карте в редакторе Godot.

## Активация плагина

1. Откройте **Project → Project Settings → Plugins**
2. Найдите плагин **"Nature Object Placer"**
3. Включите его (нажмите на переключатель **Enabled**)

После активации панель плагина появится слева в редакторе.

## Использование

1. Откройте сцену `Main.tscn` в редакторе
2. Найдите панель **"Nature Object Placer"** слева в редакторе
3. Настройте параметры:
   - **Map Size X/Z** - размер карты
   - **Min Distance** - минимальное расстояние между объектами
   - **Min Distance From Workbench** - минимальное расстояние от верстака
   - **Object Counts** - количество объектов каждого типа
   - **Object Scales** - масштабы объектов
4. Нажмите **"Place Objects"** - объекты разместятся на карте
5. При необходимости нажмите **"Clear All"** и разместите заново с другими параметрами

## Требования

Перед использованием плагина необходимо:
1. Запустить **Project → Tools → Create Nature Object Scenes** для создания отдельных сцен объектов
2. Убедиться, что папка `res://scenes/nature_objects/` содержит `.tscn` файлы объектов
